# Infrastructure / Config

Esta carpeta contiene la configuración de infraestructura para el microservicio **payments**.

## Propósito

Archivos de configuración necesarios para la operación del microservicio, incluyendo:

- Configuración de acceso a bases de datos (por ejemplo, strings de conexión, credenciales)
- Configuración de mensajería (por ejemplo, parámetros de conexión a Kafka, nombres de topics)
